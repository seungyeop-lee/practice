# 토비의 스프링 3.0 실습

- "[토비의 스프링 3.0](http://acornpub.co.kr/book/toby-spring3)" 책을 실습한 코드를 업로드 한다.

## 실습 방법

- 각 소단원 별로 커밋한다.
- 실습에 사용된 SQL문도 같이 포함시킨다.
- 개인적으로 설명이 필요하다고 느끼는 부분은 주석을 단다.

## 실습 환경

- JDK: AdoptOpenJDK 8
    - version: jdk8u212-b03
- IDE: Spring Tool Suite 3
    - Version: 3.9.8
- DataBase: MySQL
    - Version: 8.0.16 Community
