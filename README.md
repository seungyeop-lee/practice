# 처음 배우는 스프링 부트 2 실습

- "[처음 배우는 스프링 부트 2](http://www.hanbit.co.kr/store/books/look.php?p_code=B4458049183)" 책을 실습한 코드를 업로드 한다.

## 실습 방법

- 각 소단원 별로 커밋한다.
- 개인적으로 설명이 필요하다고 느끼는 부분은 주석을 단다.

## 실습 환경

- JDK: AdoptOpenJDK 11 (HotSpot JVM)
    - version: 11.0.3_7
- IDE: IntelliJ Ultimate
    - Version: 2019.1.3
