# Groovy In Action 실습

- "[Groovy In Action, 2nd edition](https://www.manning.com/books/groovy-in-action-second-edition)" 책을 실습한 코드를 업로드 한다.

## 실습 방법

- 각 소단원 별로 커밋한다.
- 개인적으로 설명이 필요하다고 느끼는 부분은 주석을 단다.

## 실습 환경

- JDK: AdoptOpenJDK 8 (HotSpot JVM)
    - version: 8.0.212_04
- IDE: IntelliJ Ultimate
    - Version: 2019.2
